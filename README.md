# Jquery-UI
---
### What is jQuery UI?
jQuery UI is a curated set of user interface interactions, effects, widgets, and themes built on top of the jQuery JavaScript Library.
---
This project contains 10 **Jquery-UI** such as:
1. Autocomplete
2. Accordion
3. Progressbar
4. Slider
5. Tabs
6. Tooltip
7. Draggable
8. Droppable
9. Dialog
10. Selectable
